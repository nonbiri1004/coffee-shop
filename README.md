# ☕ Coffee Shop — 響應式咖啡品牌電商網站

一個以咖啡品牌為主題的響應式電商網站,從版面設計到前端互動皆獨立完成。

## 🔗 Demo
👉 https://nonbiri1004.github.io/coffee-shop/

## 🛠 使用技術
- HTML5 / CSS3(自訂 CSS 變數主題)
- Bootstrap 5(RWD 響應式設計)
- JavaScript (ES6)

## ✨ 主要功能
- 響應式版面,支援桌機與行動裝置
- 首頁輪播、產品分類與商品列表
- 購物側欄(Offcanvas)
- 捲動特效、回頂按鈕
- 聯絡表單與前端驗證

## ⚡ 效能優化
- 圖片經壓縮優化,整體資源由 51MB 降至約 4MB,大幅提升載入速度
